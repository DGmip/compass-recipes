compass-recipes

Sass recipes, using Compass. Ready to use. Send a request for commit access to add your own or just fork it and send a pull request if you want to contribute.

Recipes

  • Effects
  • Enhancements
  • Forms (skins)
  • Patterns
  • Shadows
  • Shapes
  • Shared components
  • UI components

Installation

If you are a designer, I encourage you to read the The Designerâ€™s Guide to the OSX Command Prompt Compass Recipes is available as a gem (and require Sass and Compass) 

gem update --system
gem install compass-recipes
Then in your compass configuration file, add : 
require 'compass-recipes'

Usage

Like every compass components, you just need to import file and include mixins. 

@import "recipes/shape/triangle";
.my-triangle
{
    @include triangle;
}
Like Compass does, you can include all recipes in a folder like this 
@import "recipes/shape";

.my-triangle
{
    @include triangle;
}

.my-square
{
    @include square;
}
